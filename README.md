# Trump Account Benefits Calculator

A web tool for the **Trump Account** child savings program established by the One Big Beautiful Bill Act (OBBBA). Helps parents check eligibility for federal, state, philanthropic, and employer grants and project long-term account growth.

🔗 **Live at [trumpaccountbenefits.com](https://trumpaccountbenefits.com)**

---

## Pages

| Path | What's there |
|------|--------------|
| `/` | Calculator — date of birth, ZIP, state, employer → eligibility + growth projection to age 18 / 22 / 30 / 65 |
| `/employers.html` | Searchable directory of 50 employers with known Trump Account grants or matches |
| `/states.html` | Choropleth map of all 50 states + DC showing which have announced state-level programs |
| `/privacy.html` | Privacy policy |

---

## Features

- **Grant eligibility engine** — federal seed, state programs, nationwide philanthropic grants, employer grants/matches, all evaluated from child's birth date, ZIP, state, and employer
- **ZIP income lookup** — 30,000+ ZIP codes from Census ACS data for income-based grant eligibility
- **Account growth projection** — compound growth chart and year-by-year table to age 18, 22, 30, or 65 (retirement)
- **Contribution age slider** — capped at age 17 per OBBBA rules
- **Shareable result URLs** — calculator state is hash-encoded so links can be shared
- **Email capture** — visitors can subscribe via Kit for launch notifications
- **Interactive state map** — Datawrapper-powered choropleth that auto-refreshes from a published CSV

## Trump Account Basics (OBBBA)

| Detail | Info |
|--------|------|
| Federal seed deposit | $1,000 for children born Jan 1, 2025 – Dec 31, 2028 |
| Annual contribution cap | $5,000 (family + employer combined) |
| Employer contribution sub-cap | $2,500/yr (within the $5,000 cap) |
| Contribution age window | Only while beneficiary is under 18 |
| Family contributions | After-tax in, tax-free out |
| Employer/seed/earnings | Taxed as ordinary income on withdrawal |
| Withdrawals | Prohibited before age 18 |
| Account open date | July 5, 2026 (via Form 4547 or trumpaccounts.gov) |

## Tech Stack

- **Pure HTML/CSS/JS** — no frameworks, no CDN dependencies
- **SQLite** (`trump_accounts.db`) — source of truth for employer, state grant, and ZIP income data
- **Static JS exports** (`employers.js`, `state_grants.js`, `zip_income.js`) — generated by `export.py`, loaded by frontend pages
- **Cloudflare Pages** — hosting, CDN, SSL (auto-deploys from `main`)
- **Cloudflare D1** — separate copy of the employer table that powers the directory page
- **Datawrapper** — embedded state choropleth; auto-polls `state_grants.csv` from this repo
- **Kit** — email list for launch notifications
- **Google Analytics 4** — usage analytics

## Data Management

```bash
# After updating trump_accounts.db, regenerate all derived files:
python3 export.py
#   → employers.js, state_grants.js, zip_income.js
#   → state_grants.csv (Datawrapper data source)
#   → *_seed.sql (re-seedable SQL dumps)

# Import Census ACS ZIP income data (one-time):
python3 census_import.py

# Ingest employer/state grant data from text via Claude API:
export ANTHROPIC_API_KEY=your_key_here
python3 ingest.py
```

### Updating the employer directory on Cloudflare D1

The `/employers.html` page reads from a Cloudflare D1 database via a Pages Function — separate from the static `employers.js`. After updating employer data:

```bash
python3 export.py  # regenerates employers_seed.sql
wrangler d1 execute trump-accounts --remote --file=employers_seed.sql
```
